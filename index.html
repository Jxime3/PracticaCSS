CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es el lenguaje estándar utilizado en el desarrollo web para definir la presentación visual de documentos estructurados con HTML o XML. Controla aspectos como colores, fuentes, diseño, espaciado y posicionamiento, separando el contenido de la estructura.
